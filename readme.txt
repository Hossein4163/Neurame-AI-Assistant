== Description ==

A smart AI-powered assistant to recommend educational courses for children based on their interests and parental goals.
Built with Elementor widgets, WooCommerce compatibility, Gemini 2.0 Flash API integration, and Tailwind CSS.

== Features ==

* AI-Powered Course Recommendation
* Child Management (Parental Mode)
* Trainer Reports and Analytics
* Elementor Widget Support
* Fully Translatable (fa_IR and en_US)
* Responsive Design using Tailwind CSS
* Google Gemini 2 Flash Integration

== Installation ==

1. Upload `neurame-ai-assistant.zip` via WordPress admin panel > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Configure API Key and settings from **Neurame AI Assistant** menu.
