A smart AI assistant for WooCommerce-based educational platforms, helping parents and trainers manage children's progress and course recommendations.

== Description ==

Neurame AI Assistant is a WordPress plugin designed to help parents and trainers:
- Manage children's profiles (age, interests, goals)
- Track training progress via trainer reports
- Receive AI-based course recommendations
- Support both parental and individual modes
- WooCommerce-compatible

== Features ==
* Parental dashboard for managing children
* Trainer reporting system with AI-assisted rewriting
* Smart chart visualization of child's progress
* ChatGPT or Gemini integration
* Secure AJAX handling
* WooCommerce integration for course detection
* Persian RTL support (Farsi language)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/neurame-ai-assistant/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WooCommerce → My Account → Smart Dashboard
